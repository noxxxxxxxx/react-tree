# 教程

# 快速入门

前提条件

  • 熟悉 React 和 Redux 语法
  • 已安装 16.0 或更高版本的 Node.js

# 安装

根据你的包管理工具，选择下列任意一项命令执行。

使用 NPM:

$ npm i figma-tree

使用 Yarn:

$ yarn add figma-tree

使用 PNPM:

$ pnpm i figma-tree

# 导入

将 figma-tree 组件导入，设置数据和默认参数并进行渲染。

import Tree from 'figma-tree'
import 'figma-tree/dist/style.css'

export default App () => {
  const data = []
  const config = {}

  return (
    return (
    <Tree
      {...config}
      data={data}
    />
  )
  )
}
